# 工作表导出/导入 .xlsx(带样式)

## 目标
导出/导入 .xlsx 时**保留单元格样式**(颜色、字体、边框、对齐、合并、列宽、行高、数字格式),WPS/Excel 打开带格式,导入尽量还原。

## 技术方案
- **依赖**:移除 `xlsx`,加 `exceljs`(读写带样式 xlsx;SheetJS 社区版写 xlsx 不带样式,故换库)。
- **导出**:`fwb.save()` 拿完整 snapshot -> 解析 `styles`/`cellData`/`mergeData`/`columnData`/`rowData` -> 映射到 exceljs 单元格样式 -> `writeBuffer` 下载。
- **导入**:exceljs 读 -> 构建 `IWorkbookData` snapshot(`styles` 样式表 + `cellData` + `mergeData` + `columnData` + `rowData`) -> dispose 旧 Univer + `createWorkbook(新snapshot)` -> `fwb` 指向新工作簿 -> 用户点保存同步。重建 snapshot 最干净,无残留旧样式。

## 样式映射(Univer IStyleData <-> exceljs)
- `bl`(0/1) <-> `font.bold`
- `it`(0/1) <-> `font.italic`
- `ul{ s:1 }` <-> `font.underline='single'`;`st{ s:1 }` <-> `font.strike`
- `fs` <-> `font.size`;`ff` <-> `font.name`
- `cl{ rgb:'#RRGGBB' }` <-> `font.color.argb='FFRRGGBB'`
- `bg{ rgb }` <-> `fill={type:'pattern',pattern:'solid',fgColor:{argb:'FFRRGGBB'}}`
- `bd{t,r,b,l:{ s:BorderStyleTypes数字, cl:{rgb} }}` <-> `border.{top,bottom,left,right}{style, color:{argb}}`。BorderStyleTypes 映射:THIN=1->'thin',HAIR=2->'hair',DOTTED=3->'dotted',DASHED=4->'dashed',DOUBLE=7->'double',MEDIUM=8->'medium',THICK=13->'thick',其余->'thin'
- `ht`(1=LEFT,2=CENTER,3=RIGHT,4=JUSTIFIED) <-> `alignment.horizontal`('left'/'center'/'right'/'justify')
- `vt`(1=TOP,2=MIDDLE,3=BOTTOM) <-> `alignment.vertical`('top'/'middle'/'bottom')
- `tb`(3=WRAP) <-> `alignment.wrapText=true`
- `tr{ a:角度 }` <-> `alignment.textRotation`
- `n{ pattern }` <-> `cell.numFmt`

## 单位换算
- 颜色:`#RRGGBB` <-> `FFRRGGBB`(加 FF alpha)
- 列宽:Univer 像素 <-> exceljs 字符宽度。`width ≈ (px-5)/7`,`px ≈ width*7+5`
- 行高:Univer 像素 <-> exceljs points。`pt = px/1.333`,`px = pt*1.333`

## 文件改动
1. `web/package.json` - 移除 `xlsx`,加 `exceljs`
2. `web/src/sheet-io.ts` - 重写:
   - `exportWorkbookToXlsx(fwb, filename)`:`fwb.save()` -> 遍历 snapshot sheets -> 映射值+样式+合并+列宽+行高到 exceljs -> 下载
   - `importXlsxToWorkbook(data, template, title)`:exceljs 读 -> 构建 IWorkbookData snapshot(样式去重入 styles 表) -> 返回 snapshot
3. `web/src/views/Workbook.vue`:
   - 加 `mountWithSnapshot(snapshot)`:dispose 旧 + setupUniver + createWorkbook + 设编辑态
   - `onImportFile` 改:读 arrayBuffer -> `importXlsxToWorkbook` 拿 snapshot -> `mountWithSnapshot` -> 提示保存

## 处理范围(保留)
值、字体(颜色/粗体/斜体/下划线/删除线/字号/字体名)、背景色、边框、对齐、换行、文字旋转、合并单元格、列宽、行高、数字格式。公式导出为值(与 extractForSync 一致)。

## 不处理(限制,保证稳定)
条件格式、富文本(转纯文本)、Univer 特有(padding/textDirection/上下标/主题色 th)。这些 WPS/Excel 也不完全对等。

## 验证
1. `npm install` 装 exceljs,`npm run build` 通过。
2. 给某店工作簿单元格设颜色/加粗/边框/合并 -> 导出 xlsx -> WPS/Excel 打开看到样式。
3. WPS 改颜色/加行 -> 导入 -> 画布显示样式 -> 保存同步 -> 数据大屏更新。
4. 列宽/行高 round-trip 检查。
