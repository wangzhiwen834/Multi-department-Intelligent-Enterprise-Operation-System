# @univerjs/telemetry

[![npm version](https://img.shields.io/npm/v/@univerjs/telemetry?style=flat-square)](https://npmjs.com/package/@univerjs/telemetry)
[![license](https://img.shields.io/npm/l/@univerjs/telemetry?style=flat-square)](https://npmjs.com/package/@univerjs/telemetry)
[![downloads](https://img.shields.io/npm/dm/@univerjs/telemetry?style=flat-square)](https://npmjs.com/package/@univerjs/telemetry)

`@univerjs/telemetry` defines the telemetry service interface used by Univer packages. Applications can implement and register this service to collect usage or diagnostic events.

## Package Overview

| Package | UMD global | CSS | Locales | Facade entry |
| --- | --- | :---: | :---: | :---: |
| `@univerjs/telemetry` | `UniverTelemetry` | No | No | No |

## Installation

```sh
pnpm add @univerjs/telemetry
# or
npm install @univerjs/telemetry
```

Keep all `@univerjs/*` packages on the same version.

## Usage

```ts
import { ITelemetryService } from '@univerjs/telemetry';
```
