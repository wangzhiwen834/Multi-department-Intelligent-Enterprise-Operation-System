`@univerjs/preset-sheets-data-validation` provides a preset for data validation in Univer Sheets.

## Package Overview

| Package                                   | CSS | Locales | Facade entry |
| ----------------------------------------- | :-: | :-----: | :----------: |
| `@univerjs/preset-sheets-data-validation` | Yes |   Yes   |     Yes      |

## Installation

```sh
pnpm add @univerjs/preset-sheets-data-validation
# or
npm install @univerjs/preset-sheets-data-validation
```

Keep all `@univerjs/*` packages on the same version.

## Usage

```ts
import { UniverSheetsDataValidationPreset } from '@univerjs/preset-sheets-data-validation';

// Use with createUniver:
// createUniver({ presets: [UniverSheetsDataValidationPreset()] });
```
