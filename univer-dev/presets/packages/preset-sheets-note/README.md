`@univerjs/preset-sheets-note` provides a preset for notes in Univer Sheets.

## Package Overview

| Package                        | CSS | Locales | Facade entry |
| ------------------------------ | :-: | :-----: | :----------: |
| `@univerjs/preset-sheets-note` | Yes |   Yes   |     Yes      |

## Installation

```sh
pnpm add @univerjs/preset-sheets-note
# or
npm install @univerjs/preset-sheets-note
```

Keep all `@univerjs/*` packages on the same version.

## Usage

```ts
import { UniverSheetsNotePreset } from '@univerjs/preset-sheets-note';

// Use with createUniver:
// createUniver({ presets: [UniverSheetsNotePreset()] });
```
