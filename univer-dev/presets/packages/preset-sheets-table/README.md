`@univerjs/preset-sheets-table` provides a preset for table features in Univer Sheets.

## Package Overview

| Package                         | CSS | Locales | Facade entry |
| ------------------------------- | :-: | :-----: | :----------: |
| `@univerjs/preset-sheets-table` | Yes |   Yes   |     Yes      |

## Installation

```sh
pnpm add @univerjs/preset-sheets-table
# or
npm install @univerjs/preset-sheets-table
```

Keep all `@univerjs/*` packages on the same version.

## Usage

```ts
import { UniverSheetsTablePreset } from '@univerjs/preset-sheets-table';

// Use with createUniver:
// createUniver({ presets: [UniverSheetsTablePreset()] });
```
