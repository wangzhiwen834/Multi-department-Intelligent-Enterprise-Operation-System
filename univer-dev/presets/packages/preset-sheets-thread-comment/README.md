`@univerjs/preset-sheets-thread-comment` provides a preset for thread comments in Univer Sheets.

## Package Overview

| Package                                  | CSS | Locales | Facade entry |
| ---------------------------------------- | :-: | :-----: | :----------: |
| `@univerjs/preset-sheets-thread-comment` | Yes |   Yes   |     Yes      |

## Installation

```sh
pnpm add @univerjs/preset-sheets-thread-comment
# or
npm install @univerjs/preset-sheets-thread-comment
```

Keep all `@univerjs/*` packages on the same version.

## Usage

```ts
import { UniverSheetsThreadCommentPreset } from '@univerjs/preset-sheets-thread-comment';

// Use with createUniver:
// createUniver({ presets: [UniverSheetsThreadCommentPreset()] });
```
