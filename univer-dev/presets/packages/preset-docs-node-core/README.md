`@univerjs/preset-docs-node-core` provides a preset for running Univer Docs on Node.js with core features.

## Package Overview

| Package                           | CSS | Locales | Facade entry |
| --------------------------------- | :-: | :-----: | :----------: |
| `@univerjs/preset-docs-node-core` | No  |   Yes   |     Yes      |

## Installation

```sh
pnpm add @univerjs/preset-docs-node-core
# or
npm install @univerjs/preset-docs-node-core
```

Keep all `@univerjs/*` packages on the same version.

## Usage

```ts
import { UniverDocsNodeCorePreset } from '@univerjs/preset-docs-node-core';

// Use with createUniver:
// createUniver({ presets: [UniverDocsNodeCorePreset()] });
```
